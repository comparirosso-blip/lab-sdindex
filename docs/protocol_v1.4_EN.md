## I. System Architecture: Three-Layer Model

Observers must strictly distinguish between data layers to avoid conflating *sensory facts* with *subjective interpretation*.

| Layer | Objects | Function | Requirement |
| :---- | :---- | :---- | :---- |
| **Layer 1 — Measurement** | Five sensory vectors $V_s$ ($V_t, V_l, V_r, V_g, V_e$) and $V_{e\_valence}$ | Capture physical and neurological trigger intensity | Must be first-hand and uncontaminated |
| **Layer 2 — Semantic** | Subtypes, Free Keywords, Relation Tags | Define structural attributes of sensory sources and cross-site logical linkages | — |
| **Layer 3 — Narrative** | Field Notes, Emotion Trajectory, Key Insight | Record emotional dynamics, turning points, and research findings | — |

## II. Five Sensory Vectors ($V_s$) — Quantification Reference Matrix (1.0–10.0)

This matrix is the core calibration tool for scoring $V_s$. Observers assess the depth of sensory activation at the site.

### 1. $V_t$ — Historical / Cultural Density

| Score | State | Field Description |
| :---- | :---- | :---- |
| 1.0 | Nil | Fully modern functional space. No cultural reference (e.g. airport departure lounge). |
| 2.0 | Decorative token | Modern space with minimal decoration (e.g. a single calligraphy work in a tech lobby). |
| 3.0 | Weak cultural presence | Thematic elements appear but historical continuity is absent (e.g. chain café in a historic district). |
| 4.0 | Structural retention | *Adaptive Reuse.* Historic shell preserved; interior programme fully modernised. |
| 5.0 | Historical balance | Traditional and contemporary elements in roughly equal proportion. |
| 6.0 | Strong regional identity | Space carries pronounced locality; symbols and materials form a coherent aesthetic system. |
| 7.0 | High-density accumulation | *Architectural Heritage.* Historical layering dominates sensory experience (e.g. a century-old Zen temple outside festival season). |
| 8.0 | Active ritual presence | *Intangible Practice.* Live ritual activity in progress (e.g. tea ceremony, religious observance). |
| 9.0 | Near-pure cultural state | Almost no contemporary intrusion; every detail conforms to traditional philosophical logic (e.g. Katsura Imperial Villa). |
| 10.0 | Absolute field | Sacred or absolute authority renders the observer's modern identity entirely subordinate. |

### 2. $V_l$ — Light Texture

| Score | State | Field Description |
| :---- | :---- | :---- |
| 1.0 | Uniform artificial | Even, high-brightness modern LED. No shadow, no gradation, full transparency. |
| 2.0 | Standard commercial | Typical office lighting; functional emphasis, minimal light–shadow variation. |
| 3.0 | Mixed and incoherent | Shopping-mall lighting. Natural and artificial light mix randomly; contrast low and blurred. |
| 4.0 | Directional guidance | Light begins to define spatial boundaries with intentional directionality. |
| 5.0 | Functional balance | *Artificial Neutral.* Layered artificial light with legible shadow edges. |
| 6.0 | Natural light incision | Large-area natural light in modern architecture creates clear bright/dark zones. |
| 7.0 | Natural diffused layering | *Natural Diffused.* Filtered soft light; shadows are warm and have depth. |
| 8.0 | High-register *In'ei* | Traditional screens and lattices create multiple shadow layers; transparency shifts with viewing angle. |
| 9.0 | Extreme contrast | *Candlelight / Natural Direct.* Light becomes the spatial protagonist (as described in *In Praise of Shadows*). |
| 10.0 | Sacred light domain | Absolute natural light control; precisely marks the intersection of time and space. |

### 3. $V_r$ — Texture & Tactility

| Score | State | Field Description |
| :---- | :---- | :---- |
| 1.0 | Industrial alienation | Plastic, laminate, cheap metal. Cold to the touch; body and environment maximally distant. |
| 2.0 | Low-grade mass production | Rough industrial paint or composite board; no tactile detail. |
| 3.0 | Mass-produced refinement | Standard hotel materials. Smooth but generic; no impulse to engage. |
| 4.0 | Commercial quality | Good quality natural or engineered stone; initial sense of materiality. |
| 5.0 | Material awakening | *Architectural Design.* Unfinished wood or stone; texture is legible and warm. |
| 6.0 | Aged surface quality | Light *patina* visible on surfaces; material detail begins to invite the eye to linger. |
| 7.0 | Restrained tactility | *Social Protocol.* Vessels and furniture embody a highly disciplined aesthetic. |
| 8.0 | High-resonance contact | Touching the material (e.g. a wooden handrail) produces a strong sense of quality; body distance closes. |
| 9.0 | Existential permission | *Existential Permission.* Material carries a profound sense of containment (e.g. the warmth of ancient tatami). |
| 10.0 | Zero-distance communion | Material, temperature, and humidity reach absolute harmony; the body dissolves into the environment. |

### 4. $V_g$ — Spatial Geometry

| Score | State | Field Description |
| :---- | :---- | :---- |
| 1.0 | Chaotic | Blurred boundaries, clutter. No visual guidance; geometric logic fragmented. |
| 2.0 | Random accumulation | Space lacks order; objects and walls relate randomly. |
| 3.0 | Functional partition | Standard compartmentalisation. Boundaries clear but purely functional; basic rectangular stacking. |
| 4.0 | Modern geometry | Clear boundaries, single visual axis; minimalist modernist order. |
| 5.0 | Rhythmic order | *Agricultural Rhythmic.* Order derived from repeated land-use rhythm (e.g. vineyard rows, terraced fields). |
| 6.0 | Guided structure | Geometric corridor or axis strongly draws the gaze in a single direction. |
| 7.0 | Deep enclosure | *Enclosure Deep.* Boundaries precisely layered; implies an interior core. |
| 8.0 | High geometric order | Traditional architectural axes and module grids legible; geometry and ritual sense are coupled. |
| 9.0 | Geometric strictness | *Geometric Strict.* Absolute geometric order; visual guidance carries physical compulsion. |
| 10.0 | Void-centred | *Void Centred.* Geometry exists entirely to define the emptiness at the centre. |

### 5. $V_e$ — Temporal Intensity

| Score | State | Field Description |
| :---- | :---- | :---- |
| 1.0 | Clock-synchronised | Time perception aligned with the watch. High information density, high mobility. |
| 2.0 | Mild interference | A quiet urban corner; rhythm slightly slower, but still within ordinary time. |
| 3.0 | Minor deviation | Ambient sound or phenological change occasionally draws attention to time passing. |
| 4.0 | Contemplative stillness | Space is quiet; observer begins turning attention inward. |
| 5.0 | Narrative intervention | *External Narrative.* Historical background or AI activates a temporal shift. |
| 6.0 | Reality detachment | Strong spatial heterogeneity produces a temporary sensation of leaving ordinary time. |
| 7.0 | Temporal arrest | *Intrinsic Spatial.* Spatial structure (e.g. dry garden) causes time perception to expand or stop. |
| 8.0 | Deep expansion | Breathing and heartbeat rhythm change; one minute feels extended to several. |
| 9.0 | Multi-layered superposition | *Self-Time Projection.* Observer senses simultaneous existence across multiple historical time layers. |
| 10.0 | Time disappears | Senses fully absorbed. The boundary between self and time dissolves; pure present. |

## III. $V_{e\_valence}$ — Temporal Valence Reference Table (−1.0 to +1.0)

$V_{e\_valence}$ defines the *quality* of temporal shift. **Required when $V_e > 0$; recommended otherwise.**

| Value | Label | Judgement Phrase | Field Description |
| :---- | :---- | :---- | :---- |
| +1.0 | Positive Expansion | "I want to stay longer" | Time feels held and still; accompanied by deep relaxation, resonance, or quiet. |
| +0.5 | Mildly Positive | "I'd like to linger a little" | Light pleasant extension; space rhythm feels slower than the outside world. |
| 0.0 | Neutral / Mixed | "Time feels different" | Highly informatised or ritualised context; time is perceived but carries no emotional valence. |
| −0.5 | Mildly Negative | "Slightly heavy" | Time is sensed, accompanied by mild fatigue or impatience. |
| −1.0 | Negative Compression | "The longer I stay, the worse I feel" | Time perceived as burden, pressure, or entrapment; often co-occurs with sensory saturation. |

## IV. Interpretive Disambiguation: Geometry and Relation Tags *(Living Document)*

This section expands continuously as the corpus grows. It calibrates boundary cases.

- **Void Centred vs. Enclosure Deep**: The focal point is *the central emptiness* vs. *a solid entity deeper in the space*.
- **Preloaded Knowledge vs. Active Information Retrieval**: Background knowledge acquired before entry vs. information actively retrieved during the visit (AI / Google / website).
- **Temporal Variant vs. Sensory Mirror**: Variant records of the *same site at different times* vs. *different sites sharing similar phenomenological structure*.
- **[Case: Ninenzaka] Re-coding divergence**: When a record is re-coded from memory under a newer schema version without access to the original, the two versions must be treated as independent data points—not as an upgraded replacement. Always label with `re-coded from memory under vX.X framework` in `research_finding`.

## V. Free Keywords — Alert Logic

Alert prompts notify the observer that two simultaneously selected tags present a potential logical conflict, requiring confirmation of whether a *sensory dissociation* state is genuinely present.

### Alert A: Structural Clarity vs. Transient Overlay

| | |
| :---- | :---- |
| **Tag combination** | `off_season_clarity` + `ephemeral_density` |
| **Alert message** | *"Structural clarity and transient overlay both active? Confirm whether this reflects crowd intrusion or a specific phenological occlusion."* |
| **Co-existence note** | If both are active, the observer is likely perceiving the underlying spatial skeleton through an otherwise saturated visual field — a psychologically distinct state worth noting. |

### Alert B: Acoustic–Visual Rhythm Dissociation

| | |
| :---- | :---- |
| **Tag combination** | `acoustic_void` + `crowd_rhythm_mismatch` |
| **Alert message** | *"Acoustic void and crowd rhythm mismatch both active? Confirm whether this reflects a dissociation between visual and auditory experience."* |
| **Co-existence note** | This combination signals high material strength ($V_r$ high): the space's surfaces are absorbing sound even while visual density is elevated — a significant sensory quality marker. |

### Excluded Combination (No Alert)

| | |
| :---- | :---- |
| **Tag combination** | `diffused_clarity` + `golden_hour_compression` |
| **Rationale** | These are not physically incompatible. Diffused overcast light can co-exist with rapid colour-temperature compression at dusk. No alert is triggered; both tags may be selected simultaneously without confirmation. |

## VI. Instrument Contamination — Prohibited Practices

1. **Anchoring Effect**: Observers must not consult any AI-simulated scores before entering a site. The correct sequence is:
   > Human raw sampling → AI blind simulation → Asynchronous comparative analysis

2. **Re-coding Contamination**: When retroactively completing a record from memory under a newer schema version, without access to the original version, the two records **must not** be treated as the same entry. Both versions must be preserved, with the re-coded version labelled:
   > `re-coded from memory under vX.X framework, YYYY-MM-DD`

## VII. Standard Execution Workflow

| Step | Action | Notes |
| :---- | :---- | :---- |
| **Step 1** | **Entry State** | Log current physiological and psychological state before entering. |
| **Step 2** | **Raw Sensing** | Score $V_t \rightarrow V_l \rightarrow V_r \rightarrow V_g \rightarrow V_e$ using the sliders. |
| **Step 3** | **$V_{e\_valence}$ Assessment** | If $V_e > 0$, assess and record temporal valence. |
| **Step 4** | **Semantic Linkage** | Select `free_keywords`; confirm any triggered alert prompts. |
| **Step 5** | **Key Insight** | Before leaving the site, write one structurally meaningful sentence capturing the core observation. |
